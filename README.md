# 💰 家計簿Webアプリ

シンプルで使いやすい家計簿Webアプリです。ブラウザだけで動作し、データはすべてローカル (あなたのPC) に保存されます。

## 📁 ファイル構成

```
kakeibo-app/
├── index.html    # メインHTML
├── style.css     # スタイルシート
├── app.js        # アプリケーションロジック
└── README.md     # このファイル
```

## 🚀 使い方

### 1. アプリを開く

**方法1: ダブルクリック**
- `index.html` をダブルクリックしてブラウザで開く

**方法2: ドラッグ&ドロップ**
- `index.html` をブラウザにドラッグ&ドロップ

**方法3: ターミナルから**
```bash
cd ~/kakeibo-app
open index.html
```

### 2. 支出を記録する

1. 「入力」タブで情報を入力
2. 日付、カテゴリ、金額は必須
3. 場所、商品名・メモは任意
4. 「記録する」ボタンをクリック

### 3. 支出を確認・編集する

1. 「一覧」タブで全支出を確認
2. 月で絞り込み可能
3. 「編集」ボタンで内容を変更
4. 「削除」ボタンで削除

### 4. 分析する

1. 「分析」タブを開く
2. 年・月を選択
3. カテゴリ別集計とグラフで可視化

## 📊 機能

### ✅ 実装済み機能

- 支出の入力・編集・削除

## 💾 データの保存

- すべてのデータはブラウザの **localStorage** に保存されます
- インターネット接続不要
- データは外部に送信されません
- 同じブラウザ・同じPCでのみアクセス可能

### ⚠️ 注意事項

- ブラウザのデータを消去すると、家計簿データも消えます
- 定期的にエクスポート機能でバックアップを取ることをおすすめします

## 📤 データのバックアップ

### エクスポート方法
1. 右上の「データエクスポート」ボタンをクリック
2. CSVファイルがダウンロードされます
3. このファイルを安全な場所に保存

### インポート方法
1. 右上の「データインポート」ボタンをクリック
2. 以前エクスポートしたCSVファイルを選択
3. データが復元されます

## 🔧 カスタマイズ

### カテゴリを追加・変更

`index.html` と `app.js` の以下の部分を編集:

**index.html (2箇所)**
```html
<select id="category">
    <option value="新しいカテゴリ">新しいカテゴリ</option>
    <!-- 追加 -->
</select>
```

### 色を変更

`style.css` の以下の部分を編集:

```css
/* メインカラー */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

好きな色に変更できます。

## 🐛 トラブルシューティング

### データが表示されない
- ブラウザのキャッシュをクリアしてみてください
- 別のブラウザで開いた場合、データは共有されません

### グラフが表示されない
- インターネット接続を確認してください (Chart.js読み込みに必要)
- ブラウザを最新版にアップデートしてください

### エクスポートできない
- ポップアップブロッカーを無効にしてください
